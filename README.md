# Task Management API

## Description
A backend application built using ASP.NET Core and SQL Server for basic managing tasks.

## Features
- Create, update, delete tasks(CRUD)
- User authentication using JWT
- User registration and login
- Role selection during registration (Admin or User)

## Technologies
- C#
-ASP.NET Core Web API
- SQL Server
- Entity Framework Core

## How to Run
1. Clone the repository
2. Open in Visual Studio
3. Run the project
4. Open Swagger: https://localhost:7298/swagger/index.html
